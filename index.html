Welcome to ShooymCord

A local peer-to-peer chat system that runs entirely in your browser.

IMPORTANT: The "Server Host" must be open on one computer for the chat to work. If the Host closes, the chat stops for everyone.

Server Host

Open this on ONE computer only. This acts as the "Tower" that connects everyone.

ShooymCord

Open this to chat! Use this on all other computers to join the room.
